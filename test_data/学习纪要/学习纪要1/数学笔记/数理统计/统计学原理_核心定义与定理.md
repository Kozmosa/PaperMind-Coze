## 核心定义汇总

### 2.1 抽样与数据

| 编号 | 定义 | 作用 |
|------|------|------|
| **Def 2.1** | **有限总体 (Finite Population)**：总体 $U = \{u_1, u_2, ..., u_N\}$ | 作为抽样框架的基础概念 |
| **Def 2.2** | **变量 (Variable)**：$X: U \rightarrow \mathcal{X}$，将每个单元映射到取值空间 | 定义我们要研究的对象 |
| **Def 2.3** | **样本 (Sample)**：$S = (u_1, ..., u_n)$，从总体中抽取的单元集合 | 进行统计分析的基础 |
| **Def 2.4** | **数据矩阵 (Data Matrix)**：$X = [X_k(u_j)]_{j=1..n}^{k=1..p}$，$n \times p$ 矩阵 | 数据的组织形式 |
| **Def 2.5** | **SRSWR**：不放回简单随机抽样，$P(S = S_0) = \frac{1}{|U|^n}$ | 定义随机抽样模型 |
| **Def 2.6** | **SRSWOR**：放回简单随机抽样，$P(S = S_0) = \frac{1}{\binom{N}{n}}$，$|S_0| = n$ | 定义有限总体的抽样模型 |

### 2.3 常用分布

| 编号 | 分布 | 定义 | 关键参数 |
|------|------|------|----------|
| **Def 2.7** | **二项分布** $Bin(n,p)$ | $P(X=k) = \binom{n}{k}p^k(1-p)^{n-k}$ | $n$: 试验次数, $p$: 成功概率 |
| **Def 2.8** | **几何分布** $Geom(p)$ | $P(X=k) = (1-p)^{k-1}p$ | $p$: 成功概率 |
| **Def 2.9** | **Poisson分布** $Pois(\lambda)$ | $P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$ | $\lambda$: 强度参数 |
| **Def 2.10** | **正态分布** $N(\mu, \sigma^2)$ | $f(x) = \frac{1}{\sqrt{2\pi}\sigma}\exp(-\frac{(x-\mu)^2}{2\sigma^2})$ | $\mu$: 均值, $\sigma^2$: 方差 |
| **Def 2.11** | **Gamma分布** $Gamma(\alpha, \beta)$ | $f(x) = \frac{\beta^\alpha}{\Gamma(\alpha)}x^{\alpha-1}e^{-\beta x}$ | $\alpha$: 形状, $\beta$: 尺度 |
| **Def 2.12** | **Beta分布** $Beta(\alpha, \beta)$ | $f(x) = \frac{1}{B(\alpha,\beta)}x^{\alpha-1}(1-x)^{\beta-1}$ | $\alpha, \beta$: 形状参数 |
| **Def 2.13** | **t分布** $t_\nu$ | $T = \frac{Z}{\sqrt{U/\nu}}$，$Z\sim N(0,1)$, $U\sim\chi_\nu^2$ | $\nu$: 自由度 |
| **Def 2.14** | **F分布** $F(\nu_1, \nu_2)$ | $F = \frac{X_1/\nu_1}{X_2/\nu_2}$，$X_i\sim\chi_{\nu_i}^2$ | $\nu_1, \nu_2$: 自由度 |
| **Def 2.15** | **多元正态分布** $N_d(\mu, \Sigma)$ | $f(x) = \frac{1}{(2\pi)^{d/2}|\Sigma|^{1/2}}\exp(-\frac{1}{2}(x-\mu)'\Sigma^{-1}(x-\mu))$ | $\mu\in\mathbb{R}^d$, $\Sigma$: 协方差矩阵 |

### 2.4 经验分布与统计量

| 编号 | 定义 | 作用 |
|------|------|------|
| **Def 2.16** | **经验分布函数 (EDF)**：$\hat{F}_n(x) = \frac{1}{n}\sum_{i=1}^n 1_{\{X_i \leq x\}}$ | 用样本估计总体分布 |
| **Def 2.17** | **统计量 (Statistic)**：$T: \mathbb{R}^n \rightarrow \mathbb{R}^m$，可测函数 | 从数据中提取信息的函数 |
| **Def 2.18** | **样本均值 (Sample Mean)**：$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$ | 估计总体均值 |
| **Def 2.19** | **样本方差 (Sample Variance)**：$S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2$ | 估计总体方差（无偏） |

### 2.5 似然估计

| 编号 | 定义 | 作用 |
|------|------|------|
| **Def 2.20** | **似然函数 (Likelihood)**：$L(\theta; x) = f(x_1,...,x_n)$ | 连接数据与参数的桥梁 |
| **Def 2.21** | **对数似然函数**：$l(\theta; x) = \log L(\theta; x)$ | 便于计算导数 |
| **Def 2.22** | **极大似然估计 (MLE)**：$\hat{\theta} = \arg\max_\theta L(\theta; x)$ | 参数估计的核心方法 |
| **Def 2.23** | **似然比统计量 (LR)**：$\Lambda(x) = \frac{\sup_{\theta_0}L(\theta;x)}{\sup_{\theta\in\Theta}L(\theta;x)}$ | 假设检验的基础 |
| **Def 2.24** | **Wilks统计量**：$W(x) = -2\log\Lambda(x)$ | 渐近服从卡方分布 |

### 2.7 充分性

| 编号 | 定义 | 作用 |
|------|------|------|
| **Def 2.25** | **充分统计量 (Sufficient)**：$P_\theta(X \in A \mid T(X) = t)$ 与 $\theta$ 无关 | 包含参数所有信息的最小统计量 |
| **Def 2.26** | **指数族 (Exponential Family)**：$f(x) = h(x)\exp(\sum_j \eta_j(\theta)t_j(x) - A(\theta))$ | 重要分布族，便于找到充分统计量 |
| **Def 2.27** | **最小充分统计量**：$T$ 充分，且任意充分统计量可表示为 $T$ 的函数 | 最简洁的充分统计量 |
| **Def 2.28** | **完备统计量 (Complete)**：$E_\theta[g(T)] = 0 \Rightarrow g(T) = 0$ a.s. | 保证估计量唯一性 |

## 核心定理汇总

### 2.3 分布定理

| 编号 | 定理名称 | 内容 | 作用 |
|------|----------|------|------|
| **Thm 2.1** | **正态分布标准化积分** | $\int_{-\infty}^{\infty} e^{-x^2/2}dx = \sqrt{2\pi}$ | 证明正态分布PDF归一性的基础 |
| **Thm 2.2** | **Gamma分布可加性** | $X_1\sim Gamma(\alpha_1,\beta), X_2\sim Gamma(\alpha_2,\beta) \Rightarrow X_1+X_2 \sim Gamma(\alpha_1+\alpha_2,\beta)$ | 推导 $\chi^2$ 分布的基础 |
| **Thm 2.3** | **卡方分布定义** | $Z_1,...,Z_\nu \sim_{i.i.d.} N(0,1) \Rightarrow \sum_{i=1}^\nu Z_i^2 \sim \chi_\nu^2$ | 定义卡方分布 |
| **Thm 2.4** | **Beta-Gamma关系** | $X_1\sim Gamma(\alpha_1,1), X_2\sim Gamma(\alpha_2,1) \Rightarrow \frac{X_1}{X_1+X_2} \sim Beta(\alpha_1,\alpha_2)$ | 推导Beta分布，构建t/F分布 |
| **Thm 2.5** | **t分布定义** | $Z\sim N(0,1), U\sim\chi_\nu^2 \Rightarrow T = \frac{Z}{\sqrt{U/\nu}} \sim t_\nu$ | 定义Student's t分布 |
| **Thm 2.6** | **F分布定义** | $X_1\sim\chi_{\nu_1}^2, X_2\sim\chi_{\nu_2}^2 \Rightarrow F = \frac{X_1/\nu_1}{X_2/\nu_2} \sim F(\nu_1,\nu_2)$ | 定义F分布 |
| **Thm 2.7** | **多元正态特征函数** | $X\sim N_d(\mu,\Sigma) \Rightarrow E[e^{it'X}] = \exp(it'\mu - \frac{1}{2}t'\Sigma t)$ | 证明多元正态线性变换性质 |
| **Thm 2.8** | **多元正态线性变换** | $X\sim N_d(\mu,\Sigma), A\in\mathbb{R}^{k\times d} \Rightarrow AX + b \sim N_k(A\mu+b, A\Sigma A')$ | 正态分布在线性变换下封闭 |

### 2.5 似然估计定理

| 编号 | 定理名称 | 内容 | 作用 |
|------|----------|------|------|
| **Thm 2.9** | **MLE一阶条件** | $\frac{\partial l(\theta)}{\partial\theta} = 0$ 的解 | 求MLE的基本方法 |
| **Thm 2.10** | **Bernoulli MLE** | $\hat{p} = \bar{X}$ | 给出伯努利分布的MLE |
| **Thm 2.11** | **正态分布MLE** | $\hat{\mu} = \bar{X}, \hat{\sigma}^2 = \frac{1}{n}\sum(X_i-\bar{X})^2$ | 给出正态分布的MLE |
| **Thm 2.12** | **似然比性质** | $0 \leq \Lambda(x) \leq 1, W(x) = -2\log\Lambda(x) \geq 0$ | LR统计量的基本性质 |

### 2.6 抽样分布定理

| 编号 | 定理名称 | 内容 | 作用 |
|------|----------|------|------|
| **Thm 2.13** | **弱大数定律 (WLLN)** | $X_1,...,X_n$ i.i.d., $E[X]=\mu, Var(X)=\sigma^2 < \infty \Rightarrow \bar{X} \xrightarrow{P} \mu$ | 样本均值收敛到总体均值 |
| **Thm 2.14** | **中心极限定理 (CLT)** | $\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} N(0,1)$ | 近似正态分布的基础 |
| **Thm 2.15** | **样本矩收敛** | $E[X^k] = \mu_k \Rightarrow M_k(X) \xrightarrow{P} \mu_k$ | 样本矩收敛到总体矩 |
| **Thm 2.16** | **样本方差无偏性** | $E[S^2] = \sigma^2$ | 验证无偏估计 |
| **Thm 2.17** | **正态抽样分布（精确）** | $\bar{X} \sim N(\mu,\sigma^2/n)$, $(n-1)S^2/\sigma^2 \sim \chi_{n-1}^2$, 且独立 | 正态总体精确推断的基础 |
| **Thm 2.18** | **t分布构造** | $T = \frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t_{n-1}$ | 单样本t检验的基础 |
| **Thm 2.19** | **正态参数检验** | $\frac{\sqrt{n}(\bar{X}-\mu_0)}{S} \sim t_{n-1}$ 用于 $\mu$ 检验 | 假设检验理论依据 |
| **Thm 2.20** | **两样本t检验** | $\frac{(\bar{X}-\bar{Y})-(\mu_1-\mu_2)}{S_{pooled}\sqrt{1/n+1/m}} \sim t_{n+m-2}$ | 两独立样本均值差检验 |
| **Thm 2.21** | **方差比F检验** | $\frac{S_X^2/\sigma_1^2}{S_Y^2/\sigma_2^2} \sim F_{n-1,m-1}$ | 两总体方差比较 |

### 2.7 充分性定理

| 编号 | 定理名称 | 内容 | 作用 |
|------|----------|------|------|
| **Thm 2.22** | **Neyman-Fisher因子化定理** | $T$ 充分 $\Leftrightarrow f(x) = g(T(x))h(x)$ | 判别充分统计量的充要条件 |
| **Thm 2.23** | **指数族充分统计量** | $T(X) = (\sum t_1(X_i),...,\sum t_k(X_i))$ 是充分统计量 | 指数族形式化结果 |
| **Thm 2.24** | **完备性** | $T$ 完备且充分，则 $T$ 是UMVUE的基础 | 用于证明最优估计 |
| **Thm 2.25** | **Basu定理** | $T$ 完备且充分，与任何辅助统计量独立 | 分解统计量 |

## 分布族与定理对照表

### 离散分布

| 分布 | 符号 | 参数 | MLE | 充分统计量 | 相关定理 |
|------|------|------|-----|------------|----------|
| Bernoulli | $Ber(p)$ | $p\in(0,1)$ | $\hat{p}=\bar{X}$ | $\sum X_i$ | Thm 2.10 |
| Binomial | $Bin(n,p)$ | $n\in\mathbb{N}, p\in[0,1]$ | $\hat{p}=\bar{X}/n$ | $\sum X_i$ | Thm 2.11 |
| Geometric | $Geom(p)$ | $p\in[0,1]$ | $\hat{p}=1/\bar{X}$ | $\sum X_i$ | - |
| Poisson | $Pois(\lambda)$ | $\lambda>0$ | $\hat{\lambda}=\bar{X}$ | $\sum X_i$ | Thm 2.23 |

### 连续分布

| 分布 | 符号 | 参数 | MLE | 充分统计量 | 相关定理 |
|------|------|------|-----|------------|----------|
| 正态 | $N(\mu,\sigma^2)$ | $\mu\in\mathbb{R}, \sigma^2>0$ | $\hat{\mu}=\bar{X}, \hat{\sigma}^2=S^2$ | $(\bar{X}, S^2)$ | Thm 2.11, 2.17 |
| Uniform | $U(a,b)$ | $a<b$ | $\hat{a}=X_{(1)}, \hat{b}=X_{(n)}$ | $(X_{(1)}, X_{(n)})$ | Thm 2.29 |
| Exponential | $Exp(\lambda)$ | $\lambda>0$ | $\hat{\lambda}=1/\bar{X}$ | $\sum X_i$ | 指数族 |
| Gamma | $Gamma(\alpha,\beta)$ | $\alpha>0, \beta>0$ | 数值求解 | $(\sum X_i, \sum \log X_i)$ | Thm 2.27 |
| Beta | $Beta(\alpha,\beta)$ | $\alpha>0,\beta>0$ | 数值求解 | $(\sum \log X_i, \sum \log(1-X_i))$ | - |
| t | $t_\nu$ | $\nu>0$ | - | - | Thm 2.5 |
| F | $F(\nu_1,\nu_2)$ | $\nu_1,\nu_2>0$ | - | - | Thm 2.6 |

## 重要推论速查

| 编号 | 推论 | 来源定理 |
|------|------|----------|
| **Cor 2.1** | $n\bar{X} \sim Bin(n,p)$ | Thm 2.10 |
