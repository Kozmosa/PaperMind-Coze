# Papermind 设计风格

## 产品灵魂与意象锚点

**意象锚点：** 一间洒满晨光的极简书房 —— 宽敞的胡桃木书桌，桌上摊开的笔记本、书架上的知识图谱挂画、桌面一角摆放的待处理资料托盘。

Papermind 不是冰冷的知识管理工具，而是用户专属的"第二大脑书房"——知识在这里被梳理、连接、生长，形成独一无二的知识网络。

## 气质与意象

- **关键词：** 通透、专业、治愈、生长感
- **场景感：** 清晨阳光透过白纱窗帘洒在书桌上的柔和感，知识卡片像书页一样层层叠放，质感温润但不沉重
- **情绪：** 平静、专注、有序——让学习这件事充满安全感和掌控感

## 视觉策略

### 配色方案

基于 AI/SaaS 类型扩展，调整为学习场景专属配色：

| 角色 | 色值 | 意象来源 |
|------|------|----------|
| 主色 | `#6C63FF`（薰衣草紫） | 代表"深度思考"的紫，冷静而富有创造力 |
| 主色渐变终点 | `#896BFF` | 从冷静到灵感的过渡 |
| 辅色 | `#FF6B9D`（柔和粉红） | 代表"学习热情"与"社区互动" |
| 成功/掌握 | `#00B894`（薄荷绿） | 代表"已掌握"的安心感 |
| 警告/待复习 | `#FDCB6E`（暖黄） | 代表"需要关注"的提醒 |
| 背景 | `#F0F0F3`（暖灰白） | 柔和护眼的阅读底色 |
| 卡片 | `#F0F0F3`（与背景同色） | 靠阴影区分层级，无边框 |
| 文字主 | `#2D3436` | 温润深灰，比纯黑更柔和 |
| 文字次 | `#636E72` | 中灰色辅助信息 |

### 图形语言

- 知识图谱节点使用不同颜色的圆形容器（紫色=已掌握，薄荷绿=新学，暖黄=复习中）
- 卡片全部使用大圆角（borderRadius: 24）营造"柔软感"
- 图标容器用半透明底色（主色 12% 透明度），不设描边

### 排版策略

- 使用系统默认无衬线字体（SF Pro / PingFang SC），保证跨平台一致性
- 标题：大字号（28px）+ 粗字重（800），产生强烈视觉锚点
- 正文：中等字号（15px），舒适阅读
- 辅助信息：小字号（13px），弱化视觉权重

### 动效与交互

- 所有可点击元素按下时 scale 0.97，释放回弹 1.0
- 标签/Tag 切换使用平滑淡入
- 知识图谱节点入场依次弹出，带轻微弹性
- 底部导航圆角上浮，营造"悬浮"感

### 页面结构

| 页面 | 内容构成 |
|------|----------|
| 首页 | 顶部：用户问候 + 学习进度环；中部：最近学习的知识节点横向轮播；底部：待处理草稿通知 + 快捷入口 |
| 知识库 | 顶部：搜索框；中部：知识图谱可视化区域；底部：知识节点列表（双列瀑布流卡片） |
| 社区 | 顶部：Tab 切换（好友区/社区/论坛）；中部：便利贴瀑布流；底部：发布 FAB |
| 我的 | 顶部：个人信息卡片；中部：笔记风格设置、问题日志、Reflection 入口 |

### 设计禁忌

- ❌ 不使用纯黑 #000000 文字（用深灰 #2D3436 代替）
- ❌ 不使用边框描边区分层级（用阴影代替）
- ❌ 不使用白色 #FFFFFF 作为卡片背景
- ❌ 不在卡片上使用传统阴影（使用新拟态双层阴影）
- ❌ 不使用 Emoji 作为图标（使用 @expo/vector-icons Feather 图标集）
- ❌ 不使用蓝紫渐变以外的大面积明亮色

## 技术实现

- 图标：`@expo/vector-icons`（Feather 图标集）
- 渐变：`expo-linear-gradient`
- 阴影：新拟态双层阴影（iOS 双 View 嵌套，Android elevation）
- Tab 导航：Expo Router Tabs + 自定义 TabBar（圆角上浮）
